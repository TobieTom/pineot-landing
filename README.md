# PineOT Landing Page

Professional landing page for PineOT Development Agency.

## Deployment

### Option 1: Vercel (Recommended)
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → Sign in with GitHub
3. Click "Import Project" → Select this repo
4. Click "Deploy" — done in 30 seconds
5. Add custom domain in Vercel dashboard → Settings → Domains

### Option 2: Netlify
1. Push this repo to GitHub
2. Go to [netlify.com](https://netlify.com) → Sign in with GitHub
3. Click "Add new site" → "Import an existing project"
4. Select the repo → Deploy
5. Add custom domain in Site settings → Domain management

### Option 3: GitHub Pages (Free)
1. Push to GitHub
2. Go to repo Settings → Pages
3. Set source to `main` branch, root folder
4. Your site will be live at `https://yourusername.github.io/pineot-site`

### Custom Domain Setup
After buying a domain (e.g., `pineot.tech`):
- **Vercel/Netlify**: Add domain in dashboard, update DNS records as instructed
- **GitHub Pages**: Add a `CNAME` file with your domain, update DNS
